# Channel Model

`Channel` تمثّل قناة دفع واحدة تابعة لشركة (مثلاً: e& money في الإمارات لشركة معيّنة).

الحقول المهمة:

- `company_id` – تربط القناة بالشركة المالكة.
- `provider_id` – تربط القناة بمزوّد الدفع (من جدول `payment_providers`).
- `channel_api_key` – المفتاح السري المستخدم من قبل Tasker/الموبايل لإرسال SMS إلى هذه القناة عبر `/incoming-sms`.
- `telegram_group_id` – غروب تلغرام الخاص بهذه القناة (يمكن أن يختلف من قناة لأخرى).
- `is_active` – لتفعيل/تعطيل القناة.

العلاقات:

- `Company` : كل قناة تنتمي إلى `Company` عبر `company_id`.
- `PaymentProvider` : كل قناة يمكن ربطها بمزوّد دفع عبر `provider_id`، ويمتلك المزوّد قائمة بـ `channels` عبر backref.

لوحة التحكم تستخدم هذه المعلومات لربط كل شركة بالقنوات المدعومة، وعرض القنوات المتاحة عند إعداد وسائل الدفع لشركة جديدة.
